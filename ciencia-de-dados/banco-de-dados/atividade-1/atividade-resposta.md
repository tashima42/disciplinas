### 1. Qual seria o melhor nome para esta entidade?

R: A entidade contém dados de um alvará obtido por uma empresa para atividades comerciais ou edificações, portanto, o nome "alvará" é adequado.

### 2. Indique o atributo identificador

R: O campo "NUMERO_DO_ALVARA" é o melhor candidato a identificador por ser único para cada documento.

### 3. Indique um atributo composto

R: Todos os atributos de data são compostos por ano, mês e dia. Neste conjunto de dados, um exemplo é o campo "INICIO_ATIVIDADE".

### 4. Cite exemplos de outros tipos de entidades com as quais este dado poderia ter um relacionamento

R: A criação de entidades é muito subjetiva e dependente do contexto, porém a entidade "empresa" poderia ter um relacionamento com a entidade "alvará". Outro possível relacionamento seria com a entidade "edificação"
